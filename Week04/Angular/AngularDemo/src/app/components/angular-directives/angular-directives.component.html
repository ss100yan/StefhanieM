Directives

A directive is a way that we can invoke Angular functionality within an HTML file. They act as syntax that can be used within our templates without breaking the frontend. There are several types of directives. The two types that we'll discuss are structural directives and attribute directives.

Structural Directives

The two structural directives we'll cover are "ngIf" and "ngFor".

*ngIf

What does the asterisk(*) in front of ngIf denote?

Any directive that has an asterisk (*) is a "structural directive". A structural directive is a directive that will manipulate the HTML directly. That is, it will be capable of physically deleting or adding HTML on the spot for us.

*ngFor

Since ngFor is a structural directive as well, we will need the asterisk to signify that it is a structural directive. We can use ngFor to iterate over an array of objects in order to perform some DOM manipulation a specified number of times.

Attribute Directives

Attribute directives are directives that are applied as attributtes in the elements themselves. However, these directives usually do not aim to change the actual structure of the HTML, but rather change properties such as the style of our elements.
